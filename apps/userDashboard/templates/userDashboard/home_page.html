Welcome to the messaging app!

This is a simple messaging application where you can send messages back and forth along with commenting on what others have said. Keep in mind everthing you send to each other is open, so don't post anything you don't want thers to see!

Start

Manage Users

Using this application as an admin, you'll be able to add, remove, and edit users for the application.

Leave Messages

Users will be able to leave a message to another user using this application and receive comments from others as well.

Edit User Information

Admins will be able to edit another user's information (email address, first name, last name, etc.)
